4. インストール
  CsTextEditor_0_4_2.zipを解凍して、フォルダ中のCsTextEditor.exeを実行してください。
  Microsoft Windows Desktop Runtime 6.0.33を要求してくるのでインストールしてください。
  インストール後に再起動して正常に使えます。

5. アンインストール
  フォルダ毎削除してください。
